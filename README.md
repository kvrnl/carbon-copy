# CarbonCopy

**Real-time folder backup for Windows - by [KVRNL](https://kvrnl.io).**

Mirrors the folders you choose to up to two backup drives the instant anything
changes, with a safety-net sweep, recoverable deletes (per-destination Recycle
Bin), and quiet system-tray operation. One-way mirror - your sources are only
ever read, never modified.
